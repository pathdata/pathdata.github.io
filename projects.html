Computational Biology & Single-Cell Analysis

WhyNHow: Single-Cell RNA Sequencing and TCR Analytic Workflow

GitHub Repository Nature Medicine Paper

Overview: A comprehensive analytical workflow for single-cell RNA sequencing (scRNA-seq) and T-cell receptor (TCR) analysis. This project implements a reproducible pipeline for processing, quality control, and analysis of single-cell transcriptomic data, with a focus on T-cell receptor sequencing integration.

Key Features:

  • End-to-end scRNA-seq data processing pipeline
  • TCR sequencing data integration and analysis
  • Quality control metrics with mitochondrial gene assessment
  • Cell type identification and clustering
  • Reproducible R and Python workflows

Technologies: R, Python, Seurat, Single-cell analysis, TCR sequencing, Bioinformatics

Methodology

The workflow begins with raw sequencing data preprocessing, including quality filtering and normalization. Key steps include:

  1. Data Preprocessing: Raw count matrix processing, cell and gene filtering based on quality metrics
  2. Quality Control: Assessment of mitochondrial gene expression, UMI counts, and gene detection rates
  3. Normalization & Scaling: Log-normalization and variance stabilization of gene expression data
  4. Dimensionality Reduction: Principal component analysis (PCA) followed by UMAP visualization
  5. Clustering: Cell type identification using graph-based clustering algorithms
  6. TCR Integration: Integration of T-cell receptor sequencing data with transcriptomic profiles
  7. Downstream Analysis: Differential expression analysis, pathway enrichment, and clonotype characterization
Key Results

This workflow enables comprehensive analysis of single-cell transcriptomic data with integrated TCR information, facilitating:

  • Identification of distinct T-cell subsets based on both transcriptomic and TCR profiles
  • Characterization of clonal expansion patterns in T-cell populations
  • Assessment of cellular diversity and heterogeneity in complex tissue samples
  • Integration with clinical phenotypes for biomarker discovery
